Membuat Komponen Vue

Cara mengekspor dan menggunakan komponen tersebut:

tes.vue

 <template>

  <h1>Hello from a component</h1>

</template>


<script>

export default {

  name: 'TestComponent',

  // Data, methods, computed properties, etc. for your component

}

</script>

Dalam contoh di atas, kita menambahkan export default untuk mengexport komponen dengan nama "TestComponent".

Kita dapat mengimpor komponen tersebut seperti ini:

app.vue

 <template>

  <div>

    <h2>My App</h2>

    <TestComponent /> <!-- Ini adalah penggunaan komponen yang diimpor -->

  </div>

</template>


<script>

import TestComponent from '@/components/test.vue'; // Pastikan path sesuai dengan struktur proyek Anda

export default {

  name: 'App',

  components: {

    TestComponent // Daftarkan komponen yang diimpor di sini

  }

  // Data, methods, computed properties, dll. untuk komponen App

}

</script>
